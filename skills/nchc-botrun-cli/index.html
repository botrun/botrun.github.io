nchc-botrun-cli

國網中心波特人 Agentic AI 孵化器。當用戶提到「國網排行榜」「NCHC leaderboard」「國網中心評測」時使用此技能。

NCHC-Botrun - 國網中心波特人 Agentic AI 孵化器

快速指令

/hi-direct - Direct 壓力測試

平行呼叫所有打星號模型，測量：
- TTFT (Time To First Token) - 首 token 回應時間
- 總時間 - 完整回應時間
- Tokens/sec - 每秒生成速度

執行方式：

`bash
cd projects/nchc-botrun-cli && sh/cli.sh --hi-direct
`

輸出結果儲存於 data/hi-direct-results.json

/hi-agentic - Agentic AI 壓力測試

`bash
cd projects/nchc-botrun-cli && sh/cli.sh --hi-agentic
`

輸出結果儲存於 data/hi-agentic-results.json

可用模型

1. TAIDE 12B（臺灣本土）- Gemma-3-TAIDE-12b-Chat
2. Llama 4 Maverick 17B
3. Devstral 2 123B
4. Devstral Small 24B
5. Ministral 3 14B
6. Ministral 3 8B
7. GPT-OSS 120B

CLI 命令

| 命令 | 說明 |
|------|------|
| sh/cli.sh | 互動模式（預設） |
| sh/cli.sh --hi-direct | Direct 壓力測試 |
| sh/cli.sh --hi-agentic | Agentic AI 壓力測試 |
| sh/cli.sh --test | 執行單元測試 |
