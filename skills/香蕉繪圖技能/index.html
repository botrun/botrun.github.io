香蕉繪圖技能（Nano Banana 2）

> 使用 Gemini 3.1 Flash Image Preview 模型，4-6 秒生成高品質圖片，繁體中文渲染優異。

何時使用

- 用戶說「畫圖」「繪圖」「生成圖片」「香蕉繪圖」「banana」「畫一張」
- 用戶描述想要的圖片內容
- html-slides 技能需要情境插圖
- 任何需要 AI 生成圖片的場景

風格優先原則（必遵守）

如果用戶沒有指定風格，必須先討論風格

禁止直接跳過風格選擇就開始繪製。 先向用戶提出風格建議：

`
我準備用香蕉繪圖技能來生成圖片。在開始之前，想確認一下風格方向：

推薦以下美學風格，請選擇或告訴我你的偏好：

1. 📷 寫實攝影風（photorealistic）— 像真實照片，適合產品、場景
2. 🎨 水彩插畫風（watercolor illustration）— 柔和溫暖，適合教育、故事
3. 🖋️ 扁平設計風（flat design）— 簡潔俐落，適合簡報、UI
4. 🌸 日系動漫風（anime style）— 活潑可愛，適合年輕受眾
5. 🏛️ 油畫藝術風（oil painting）— 厚重質感，適合文化、歷史
6. ✏️ 手繪素描風（pencil sketch）— 樸實自然，適合草稿、構思
7. 🔮 賽博龐克風（cyberpunk）— 霓虹科技，適合未來、AI 主題
8. 🍃 侘寂美學風（wabi-sabi）— 不完美的美，適合哲學、禪意

或者告訴我你想要的氛圍（例如：溫暖的、專業的、科幻的），我來推薦。
`

如果用戶已指定風格，直接使用

用戶說了風格關鍵字（如「水彩」「寫實」「扁平」）就直接採用，不再詢問。

執行方式

基本繪圖

`bash
cd /Users/40gpu/coding_projects/timeless-bowen && \
shared/bh imagen generate "圖片描述 prompt, 風格描述 style" \
--aspect="1:1" \
--size="1K" \
--output="./html/output.png"
`

> DRY 原則：統一透過 shared/bh 呼叫 botrun-horse modules/imagen，不再直接呼叫 banana-draw.mjs。
> 改圖（session 延續）：shared/bh imagen edit ./前圖.png "修改描述" --output="./後圖.png"

參數說明

| 參數 | 說明 | 預設值 |
|------|------|--------|
| 第一個參數 | 圖片描述（支援繁體中文） | 必填 |
| --style, -s | 藝術風格（必須指定） | 無預設，必須先討論 |
| --aspect, -a | 長寬比 | 1:1 |
| --size | 解析度: 512px, 1K, 2K, 4K | 512px（最低，最快） |
| --output, -o | 輸出檔案路徑 | ./output.png |
| --negative, -n | 要避免的元素 | 無 |

長寬比選項

| 值 | 適用場景 |
|----|---------|
| 1:1 | 頭像、社群貼文 |
| 16:9 | 簡報、橫幅 |
| 9:16 | 手機桌布、限時動態 |
| 4:3 | 傳統投影片 |
| 3:2 | 攝影、明信片 |
| 21:9 | 超寬螢幕、電影感 |

解析度選項

| 值 | 說明 | 建議用途 |
|----|------|---------|
| 512px | 最低解析度，最快速度（預設推薦） | 草稿、預覽、簡報插圖 |
| 1K | 標準解析度（512px 失敗時 fallback） | 一般用途 |
| 2K | 高解析度 | 印刷品、海報 |
| 4K | 最高解析度 | 大型輸出 |

Prompt 工程指引

好的 prompt 結構

`
[主體] + [場景] + [光線] + [構圖] + [風格]
`

範例：
- ❌ 「一個女生」
- ✅ 「一位穿著深藍色西裝的年輕臺灣女性，站在現代辦公室的落地窗前，自然光從左側灑入，半身肖像構圖，水彩插畫風格，柔和色調」

風格 prompt 模板

| 風格 | 英文 prompt 補充 |
|------|-----------------|
| 寫實攝影 | photorealistic, natural lighting, 8K detail, shallow depth of field |
| 水彩插畫 | watercolor illustration, soft edges, gentle pastel colors, paper texture |
| 扁平設計 | flat design, geometric shapes, bold colors, minimal shadows |
| 日系動漫 | anime style, cel shading, vibrant colors, clean lines |
| 油畫 | oil painting, visible brushstrokes, rich texture, classical composition |
| 手繪素描 | pencil sketch, cross-hatching, graphite texture, white background |
| 賽博龐克 | cyberpunk, neon lights, rain-soaked streets, holographic UI |
| 侘寂 | wabi-sabi aesthetic, muted earth tones, organic imperfections, minimalist |

`

`

範例：需要 3 張圖片 → 啟動 3 個 sonnet 子代理 → 各自呼叫 shared/bh imagen generate → 全部完成後匯總

與絲滑動畫技能串接

當需要將多張圖片做成流暢動畫時：

1. 先用香蕉繪圖技能生成 keyframes（例如 4 張連續場景）
2. 再用「絲滑動畫」技能組裝：Nano Banana 2 session 延續在 keyframes 之間生成過渡幀
3. ffmpeg 最佳化組裝成 GIF（Google Chat 自動播放）

`
香蕉繪圖 → 4 張 keyframes → 絲滑動畫 → 16 張（含過渡幀）→ GIF
`

- 正確：主體描述用英文，風格和氛圍可混用中文
- 範例："A young Taiwanese woman in business suit, watercolor style, 溫暖柔和的色調"
