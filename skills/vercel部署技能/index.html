Vercel 部署技能

> 一鍵部署 HTML 到 Vercel，自動更新索引，永久記錄。

何時使用

- 用戶說「部署」「deploy」「上 Vercel」「vercel」「發佈到網路」
- html-slides 簡報完成後
- 任何 HTML 需要公開 URL

連動技能

| 技能 | 連動方式 |
|------|---------|
| html-slides（網頁投影片技能） | 簡報製作完成後 → 自動呼叫本技能部署 |
| 香蕉繪圖技能 | 圖片存在 html/ 目錄 → 與 HTML 一起打包部署 |

部署流程（必遵守）

Step 1：準備部署目錄

`bash

建立乾淨的部署目錄


PROJECT_NAME="{專案名稱}" # 英文小寫+連字號，如 three-partners
mkdir -p /tmp/${PROJECT_NAME}-deploy

複製 HTML 檔案為 index.html


cp ./html/{檔名}.html /tmp/${PROJECT_NAME}-deploy/index.html

如果有搭配圖片（香蕉繪圖技能產出），一起複製


cp ./html/{相關圖片}*.png /tmp/${PROJECT_NAME}-deploy/ 2>/dev/null
`

Step 2：部署到 Vercel

`bash
cd /tmp/${PROJECT_NAME}-deploy
vercel --yes --name ${PROJECT_NAME}
`

部署完成後記錄 URL（通常是 https://{PROJECT_NAME}.vercel.app）。

Step 3：更新索引（三處全部必更新）

#### 3a. vercel-projects.md
更新 docs/部署/vercel-projects.md：
- 在「最新活躍」區塊的最上面新增一行
- 日期格式：2026-03-16
- 說明要簡潔（一句話）

#### 3b. 首頁 web/index.html（最重要）
更新 web/index.html：
- 在

的最上面插入新卡片
- 卡片連結到 Vercel URL（target="_blank"）
- 必須標示臺灣時間戳（UTC+8），格式：2026-03-16 HH:MM UTC+8
- 用 TZ=Asia/Taipei date '+%Y-%m-%d %H:%M:%S' 取得精確時間
- 更新 header 的「更新日期」為當前臺灣時間（含年月日時分秒）

`

#### 3c. SKILL.md 技能索引
如果部署的是 SKILL.md 相關內容，在 three-skills.html 的 Vercel 頁面中同步更新。

Step 4：Git Commit + Push（必要）

`bash
cd /Users/40gpu/coding_projects/timeless-bowen
git add html/{檔名}.html html/{相關圖片}*.png docs/部署/vercel-projects.md
git commit -m "deploy: {專案名稱} to Vercel - {一句話說明}"
git push
`

HTML 是知識資產，URL 也是知識資產，兩者都必須 commit。

地雷紀錄

地雷 1：只部署不 commit — 2026-03-16

- 問題：部署到 Vercel 但忘記 commit HTML 和 URL 紀錄
- 正確：部署後必須 git add + commit + push
- 驗證：git status 確認 working tree clean

地雷 2：索引沒有更新 — 2026-03-16

- 問題：部署成功但 vercel-projects.md 沒新增紀錄
- 正確：每次部署都要在最新活躍區塊最上面新增一行
- 驗證：檢查 vercel-projects.md 最新日期是否為今天

地雷 3：圖片沒有一起部署 — 2026-03-16

- 問題：HTML 引用了圖片但部署目錄只有 index.html
- 正確：有圖片時一律 cp 到部署目錄
- 驗證：打開 Vercel URL 確認圖片顯示
