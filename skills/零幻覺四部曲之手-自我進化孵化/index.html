手（Hand）- 組織技能飛輪孵化系統

> 零幻覺四部曲第三部曲：執行、孵化、服務

---

核心使命

將任意領域知識轉化為組織技能：

1. 領域專用 CLI.ts：每個領域一個 GitHub Repo
2. 技能飛輪孵化：每次互動都自動進化
3. 快省準：5 分鐘 → 15 秒（快 20 倍、省 18 倍、準不變）
4. 耳手眼三大積累：原始記錄、組織技能、高品味題庫

---

核心指令

`bash

完整孵化流程（領域專用 CLI）


sh/cli.sh hatch run <領域> --goal 95

初始化領域 CLI Repo


sh/cli.sh hatch init <領域> --repo @botrun/<領域>-cli

孵化題庫（粗糙 → 精緻 → 巨量）


sh/cli.sh hatch quiz <領域> --level [粗糙|精緻|巨量]

優化（Opus → 8B 小模型遷移）


sh/cli.sh hatch optimize --target 8b

評測（準確度、速度、成本）


sh/cli.sh hatch benchmark

迭代進化


sh/cli.sh hatch evolve

斷點續做


sh/cli.sh hatch resume

封裝到 GitHub


sh/cli.sh hatch publish --repo
`

---

孵化器四大工具

1. 題庫生成器（generateVariations）

自動生成變化題：
- 查詢對象變化
- 查詢類型變化
- 具體指標變化
- 複合查詢變化

2. 評測器（evaluateAccuracy）

自動評測準確度：
- 執行所有測試題
- 比對標準答案
- 計算準確度
- 找出失敗案例

3. 優化器（optimizePerformance）

自動優化效能：
- 分析錯誤案例
- 生成補強題目
- 微調模型
- 重新評測

4. 迭代器（evolve）

自動迭代升級：
- 分析當前版本效能
- 識別優化機會
- 自動生成新題庫
- 產生孵化報告
- 更新版本
