耳（Ear）- 任意格式處理器

> 零幻覺四部曲之一：將任意格式轉換為 AI 可讀文字

何時使用此技能

- 用戶上傳任意格式檔案（PDF、Word、Excel、圖片、音檔、影片等）
- 用戶說「處理這個檔案」「轉成文字」「讀取這份文件」
- 用戶說「把這個 PDF 轉成 Markdown」「OCR 這張圖片」
- 用戶說「分析這份資料」「萃取文字」
- 需要將非文字格式轉換為 AI 可讀格式

核心指令

`bash

單檔處理


sh/cli.sh process

目錄批次處理


sh/cli.sh process -j 4

自然語言驅動


sh/cli.sh ask '<描述任務>'

使用特定模型


sh/cli.sh process -m claude-sonnet-4-5
`

支援格式

| 類別 | 格式 | 處理器 |
|------|------|--------|
| 音檔 | .m4a .mp3 .wav .webm .ogg .flac .aac | AudioProcessor |
| 圖片 | .png .jpg .jpeg .gif .webp .bmp .tiff .heic | ImageProcessor |
| 文件 | .pdf .docx .doc .xlsx .xls .pptx .ppt | DocumentProcessor |
| 影片 | .mov .mp4 .webm .avi .mkv .m4v | VideoProcessor |
| 資料 | .csv .json .xml .yaml .yml .toml | DataProcessor |
| 文字 | .txt .md .html .htm .rtf .tex | TextProcessor |

使用範例

處理單一 PDF


`bash
sh/cli.sh process 收集箱/文件.pdf
`

批次處理會議錄音


`bash
sh/cli.sh process 收集箱/會議錄音/ -j 4
`

OCR 圖片


`bash
sh/cli.sh process 收集箱/screenshot.png
`

轉換 Excel


`bash
sh/cli.sh process 收集箱/報表.xlsx
`

快取機制

- 使用 SHA-256 hash 識別檔案
- 預設快取 30 天
- 使用 --no-cache 強制重新處理
- 使用 sh/cli.sh cache stats 查看快取統計

輸出位置

- 預設輸出到原始檔案同目錄
- 使用 -o

指定輸出目錄
- 輸出格式：Markdown（.md）

相關檔案

- 後設提示：配置/規則/零框架後設提示-耳-任意格式處理器.yaml
- CLI 入口：sh/cli.sh
- TypeScript 模組：sh/src/

與其他技能的關係

- 永恆知己-文字轉換：本技能的精簡版，主要處理 PDF/圖片/Office
- 永恆知己-語音轉文字：專門處理會議錄音，使用 pipeline.ts
- 永恆知己-記憶：處理完成後可自動記憶重要內容
