universal-file-processor

|

永恆知己 - 萬用處理器（四部曲「眼」）

> 觸發條件：用戶需要處理任意格式檔案、生成題庫、或執行模型評測時

核心能力

1. 任意格式處理


- 文件: PDF, DOCX, XLSX, PPTX, CSV
- 圖片: PNG, JPG, GIF, WEBP (OCR + 描述)
- 音訊: M4A, MP3, WAV (語音轉文字)
- 影片: MP4, MOV, AVI (提取音軌 → 轉文字)
- 網頁: HTTP/HTTPS URL (擷取 → 整理)
- 結構化: JSON, YAML, XML

2. 高品位題庫生成


- 以人為中心（IDEO Design Thinking）
- 四種題型分佈：
- 事實性 20%
- 推論性 30%
- 應用性 30%
- 創意性 20%

3. 模型評測（九維度）


- 波特人評分法：真新全快省準人安穩
- 支援 8+ 主流模型
- 上網 vs 不上網兩批測試
- 自動生成排行榜

CLI 指令

`bash

處理檔案


sh/cli.sh process

生成題庫


sh/cli.sh quiz --count 20

執行評測


sh/cli.sh benchmark [--web]

斷點續做


sh/cli.sh resume

查看進度


sh/cli.sh status

自然語言


sh/cli.sh ask "你的指令"
`

使用流程

1. 處理檔案 → 轉換成 AI 可讀的 Markdown
2. 生成題庫 → 從內容生成高品位題目
3. 執行評測 → 多模型並行測試 + 九維度評分
4. 輸出排行榜 → Markdown + JSON 格式

九維度評分標準

| 維度 | 說明 | 10分標準 |
|------|------|----------|
| 真 | 真實性 | 完全可追溯、有明確引用來源 |
| 新 | 新穎性 | 使用最新版本、資料即時更新 |
| 全 | 完整性 | 端到端完整、無任何遺漏 |
| 快 | 速度 | < 1 分鐘完成 |
| 省 | 節省 | < 1K tokens |
| 準 | 精準度 | > 95% 正確率 |
| 人 | 以人為中心 | 完全透明、隨時可介入修正 |
| 安 | 安全性 | 全地端、零外洩風險 |
| 穩 | 穩定性 | 完全可重現、零漂移 |

認證等級：
- 🥇 金標: ≥ 85 分
- 🥈 銀標: ≥ 80 分
- 🥉 銅標: ≥ 70 分

斷點續做

所有長時間任務自動建立 Session：
- Session ID 格式: {task_type}-{YYYYMMdd-HHmmss}
- 進度存放: .cache/progress/{session_id}.json
- 隨時可用 sh/cli.sh resume 繼續
