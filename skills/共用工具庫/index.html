shared-utils

共用工具函式庫，提供文字轉換、分析、校正、雜湊快取等功能。被其他技能引用時載入。

共用工具庫

何時使用此技能

- 其他技能需要引用共用函式時
- 需要文字轉換、分析、校正功能時
- 需要雜湊快取功能時

DRY: 優先使用 shared/bh（botrun-horse）

| 功能 | shared/bh 指令 | 舊腳本（保留但不推薦） |
|------|---------------|----------------------|
| 文字轉換 | shared/bh doc text | scripts/text_converter.py |
| 雜湊快取 | botrun-horse 內建 | scripts/hash_cache.py |
| 轉錄 | shared/bh gemini transcribe | scripts/transcribe.ts |
| 文字分析 | Claude Code 內建能力 | scripts/analyze.ts |
| 勘誤校正 | Claude Code Opus 手動 | scripts/correct.ts |
| 格式化 | Claude Code 內建能力 | scripts/format.ts |

`bash

DRY 引用方式


shared/bh doc text "file.pdf"
shared/bh gemini transcribe "audio.mp3"
`
