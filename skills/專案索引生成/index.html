project-index-generator

自動掃描 projects/ 目錄並生成專案索引。當用戶說「更新專案索引」「重新生成索引」時觸發。

專案索引生成

何時使用此技能

- 用戶說「更新專案索引」
- 用戶說「重新生成索引」
- 新增專案後需要更新索引時

使用方式

`bash

Shell 版本（無依賴）


bash .claude/skills/專案索引生成/scripts/generate-project-index.sh

TypeScript 版本（需 tsx）


npx tsx .claude/skills/專案索引生成/scripts/generate-project-index.ts
`

輸出

- 索引/專案索引-自動生成.md

腳本清單

| 腳本 | 用途 |
|------|------|
| scripts/generate-project-index.sh | Shell 版本（推薦） |
| scripts/generate-project-index.ts | TypeScript 版本 |
